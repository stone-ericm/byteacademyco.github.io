`__init__`

`__init__` is run automatically when you instantiate a class.

*instantiate* means creating an "instance" of a class.

What an instance of a class?

meal VS recipe

building VS blueprint

function VS invoked function

`self`

`self` refers to the current instance of a class

`self` is the first argument to every method

`self` is how you access properties within the class definition

"properties" are the variables of an instance

"methods" are the functions of an instance

This will make a lot more sense when you start using it.
