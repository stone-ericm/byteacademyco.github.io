variable

a variable is a bucket that holds stuff

a variable is a name you make up that exists now because it holds something

dir()

>>> dir()

`dir` is a function that lists the namespace of the current scope

Namespace

The *namespace* is everything that Python knows about.

Don't worry about scope yet.

Declare and Assign

*Declare* to the computer: "This bucket exists now."

*Assign* a value into the bucket.

x = 55
x = [1, 2, 3, 4]

Python performs declaration and assignment simultaneously.

snake_case

The pythonic naming convention for variables is called "snake case".

lowercase words separated by underscores

long_variable_name = 254
name_of_a_cat = "Socks"

Flow - If / Else

your code is a river

it can fork

`if` will or will not execute code depending on the value of a boolean condition

`else` happens if `if` did not

if you need more ifs before the else, `elif`

Useful BIFS

BIF: built-in function

Python does it for you.

`len()`, `pop()`, `append()`, `split()`

For Loop

iterate through an "iterable"

a list is an iterable

for i in [1, 2, 3, 4]:
	print 2 * i

for i in range(8):
	print 15 - i

While Loop

keep going until a condition is met

i = 0
while i != 10:
	print(i)
	i += 1

make sure the condition gets met

NASA doesn't use while loops.

Functions
